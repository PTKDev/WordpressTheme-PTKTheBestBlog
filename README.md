#PTKTheBest Blog (Wordpress Theme)

PTKTheBest Blog (Wordpress Theme) is official wordpress theme of http://www.ptkthebest.eu/

## How to Install

- Copy /src content to your wp-content/themes wordpress-folder
- Upload all file with FTP
- Select theme at wp-admin control panel
